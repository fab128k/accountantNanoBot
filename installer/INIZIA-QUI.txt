WINDOWS 11
------------------------------------------------------------

  1. Scarica e decomprimi questo file ZIP
  2. Tasto destro su INSTALLA_DeepAiUG.bat
     -> "Esegui come amministratore"
  3. Segui le istruzioni a schermo

  Avvio: icona DeepAiUG sul Desktop
  Problemi: lancia CHECK_DeepAiUG.bat


MAC
------------------------------------------------------------

  1. Apri Terminale (Applicazioni -> Utility -> Terminale)
  2. Trascina installa_deepaiug_mac.sh nel Terminale e premi INVIO
     Oppure: bash installa_deepaiug_mac.sh
  3. Segui le istruzioni a schermo

  Prima apertura: tasto destro su DeepAiUG.command -> "Apri"
  Problemi: bash check_deepaiug.sh


LINUX (Ubuntu, Fedora, Arch)
------------------------------------------------------------

  1. Apri il Terminale
  2. chmod +x installa_deepaiug_linux.sh
  3. ./installa_deepaiug_linux.sh
  4. Segui le istruzioni a schermo

  Avvio: icona DeepAiUG nel menu applicazioni
         oppure: bash ~/DeepAiUG/avvia_deepaiug.sh
  Problemi: bash check_deepaiug.sh


PROBLEMI COMUNI
------------------------------------------------------------

  Cosa vedi?                 Cosa fare?
  -------------------------  ------------------------------------------
  Download lento             Normale: il modello AI pesa 2-5 GB
  Chat non risponde          Verifica che Ollama sia attivo
  Errori rossi               Lancia lo script di diagnostica per il tuo OS
  App non si avvia           Controlla la connessione Internet

  Log completo installazione in:
    Windows: C:\Users\[TUO NOME]\DeepAiUG_install_log.txt
    Mac/Linux: ~/DeepAiUG_install_log.txt


COME AGGIORNARE DEEPAIUG
------------------------------------------------------------

  Quando esce una nuova versione, NON reinstallare da zero.
  Usa lo script di aggiornamento — preserva tutte le tue
  conversazioni e configurazioni.

  WINDOWS
    1. Vai in C:\Users\[TUO NOME]\DeepAiUG\installer\
    2. Tasto destro su AGGIORNA_DeepAiUG.bat
    3. "Esegui come amministratore"
    4. Lo script aggiorna il codice e preserva le conversazioni

  MAC
    1. Apri il Terminale
    2. Scrivi: bash ~/DeepAiUG/installer/aggiorna_deepaiug_mac.sh
    3. Premi INVIO e segui le istruzioni

  LINUX
    1. Apri il terminale
    2. Scrivi: bash ~/DeepAiUG/installer/aggiorna_deepaiug_linux.sh
    3. Premi INVIO e segui le istruzioni

  COSA VIENE PRESERVATO:
    + Tutte le conversazioni salvate
    + branding.yaml (personalizzazione)
    + Configurazioni YAML (modelli, server, privacy)
    + API keys e secrets

  COSA VIENE AGGIORNATO:
    > Codice applicativo (app.py, moduli)
    > Dipendenze Python
    > Script installer

  NOTA: un backup automatico delle conversazioni viene
  salvato in ~/DeepAiUG_backup_conversations prima
  di ogni aggiornamento.


COME USCIRE DA DEEPAIUG
------------------------------------------------------------

  WINDOWS
    1. Chiudi la finestra del terminale (o premi Ctrl+C)
       Il browser puoi lasciarlo aperto o chiuderlo liberamente.
    2. Ollama resta attivo in background (normale).
       Per fermarlo completamente:
       - Cerca l'icona della pecora in basso a destra
         nella barra di Windows (system tray)
       - Tasto destro -> Quit

  MAC
    1. Torna nel Terminale e premi Ctrl+C
       Il browser puoi lasciarlo aperto o chiuderlo liberamente.
    2. Ollama resta attivo nella menu bar in alto.
       Per fermarlo completamente:
       - Clicca sull'icona Ollama nella barra in alto
       - Scegli "Quit Ollama"

  LINUX
    1. Torna nel terminale e premi Ctrl+C
       Il browser puoi lasciarlo aperto o chiuderlo liberamente.
    2. Per fermare Ollama completamente:
       - Apri un nuovo terminale e scrivi: pkill ollama
       - Oppure usa il system tray se disponibile

  NOTA: non e' necessario fermare Ollama ogni volta.
  Puoi lasciarlo attivo in background — consuma pochissime
  risorse quando non elabora richieste.
